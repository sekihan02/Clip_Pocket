# Changelog

## v0.1.0

Initial public release.

### Added

- Resident Windows clipboard monitoring for copied text.
- In-memory clipboard history with duplicate refresh behavior.
- Restore selected items to the current clipboard.
- Delete items from the history list.
- Notification-area icon with open, settings, and quit actions.
- Ctrl double-tap shortcut.
- Optional experimental right-click triple-click gesture.
- English and Japanese UI.
- Configurable retention period, including unlimited retention while running.
- Configurable maximum item count.
- Windows startup registration.
- PyInstaller-based Windows executable build.
